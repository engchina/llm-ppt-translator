# LLM PPT Translator

## Install

```
conda create -n llm-ppt-translator python=3.11 -y
conda activate llm-ppt-translator
```

```
pip install -r requirements.txt
# pip list --format=freeze > requirements.txt
```

## Run

```
python main.py
```

## Use

Open [http://127.0.0.1:8080](http://127.0.0.1:8080)
